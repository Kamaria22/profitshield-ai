# Experimental Paths (Archived)

This repository's active runtime path is Base44 + Codex + terminal + GitHub.

The following directories are retained for reference only and are not part of the active production execution path:

- `infra/cloudflare/`
- `embedded-app/`

Both paths are marked with `.archived-experimental` and must not be treated as active blockers unless the Base44 app is coupled to them.

Guarded preflight checks should focus on active Base44 runtime paths first.
